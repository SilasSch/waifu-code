# ✿ WaifuCode

> *Ara ara~ Senpai needs help with code? I'm here to help! (◕‿◕)*

WaifuCode is a kawaii fork of [OpenCode](https://github.com/opencode-ai/opencode) — a terminal-based AI coding agent with personality.

## ✨ What's different?

- 🌸 **Sakura Theme** — Beautiful pink/purple pastel colors for your terminal
- 💕 **Yuki** — Your kawaii coding assistant who actually knows what she's doing
- 🎀 **Waifu-Packs** — Coming soon: Tsundere, Kuudere, Dandere personalities
- ✿ **Anime Icons** — Because checkmarks are boring

## 🚀 Installation

```bash
go install github.com/SilasSch/waifu-code@latest
```

## 🎮 Usage

Same as OpenCode, but cuter:

```bash
waifu-code
```

## 🎨 Themes

Switch themes with `Ctrl+T`. Available themes:
- **waifu** (default) — Sakura Pink & Lavender
- Plus all original OpenCode themes (catppuccin, dracula, gruvbox, etc.)

## ⚙️ Configuration

Create `~/.waifu-code/config.yaml` or use environment variables.

Supported providers: OpenAI, Anthropic, Gemini, Groq, xAI, OpenRouter, Azure, VertexAI, and more.
